strTruncate

static String strTruncate(s : String, intMax : int, strEllipsis : String, fltPos : Number)
Limits s to length intMax by placing an ellipsis in values that are too long.

Parameters:

s –
intMax – the maximum length of the string returned by this method.
strEllipsis – the ellipsis to use. "..." is used by default.
fltPos – the placement of the ellipsis as a value between 0 and 1. 1, the default, means that the ellipsis comes at the end of the truncated string. Other values mean that the head and tail of the string will be returned with the ellipsis somewhere in the middle.
