processExts

jsx3.$Array<Object> processExts(objProcessor : jsx3.amp.ExtProc | Function, arrExt : Array<jsx3.amp.Ext>)
Processes each extension of this extension point using the visitor pattern.

Parameters:

objProcessor – the visitor. This argument may be null if the XML declaration of this extension point specifies a processor that can be constructed with the processor factory.
arrExt – optionally, a subset of the extensions of this extension point. Only these extensions will be processed if this parameter is not empty.
